Projeto FSM-PLC


A Linguagem

O projeto FSM-PLC consistiu da criação de uma nova linguagem de programação para PLCs, baseada em máquinas de estados finitos.


A IDE

A interface de programação do FSM-PLC foi um software desenvolvido em C#, contendo um compilador,um depurador,um simulador e uma aplicação para o upload do bytecode FSM para os PLCs. Aqui segue o instalador da IDE do FSM-PLC para Windows: IDE_FSMPLC_SETUP.zip


O Hardware

O FSM-PLC foi desenvolvido nas versões STM32 e PIC18, ambos rodando a máquina virtual FSM com instruções específicas da linguagem. Segue o projeto eletrônico do FSM-PLC: O hardware HARDWARE_FSMPLC_PIC.zip e o firmware FIRMWARE_FSMPLC_PIC.zip


Os Resultados

A arquitetura otimizada baseada em máquinas de estados da linguagem FSM dobrou a velocidade de execução dos programas de automação (no pior caso), quando comparada às linguagens de mercado (Ladder e FBD). A execução dos projetos também ficou mais rápida, uma vez que a linguagem FSM é mais simples e intuitiva para os engenheiros de sistemas de automação.


Projeto Arise


A Arise

A Arise foi uma empresa que criei para o desenvolvimento e fabricação de produtos eletrônicos na área de automação residencial.Infelizmente as suas atividades foram encerradas devido a crise econômica. Fomos pioneiros no Brasil na transmissão de dados pela rede elétrica para automação residencial.


A Tecnologia

Os dispositivos de automação residencial da Arise se comunicavam pela rede elétrica usando um modem e um protocolo comunicação que desenvolvi. Esse protocolo permitiu uma comunicação efetiva em todos os lugares em que foi instalado, sendo o maior deles um prédio de 14 andares. A seguir o link de alguns manuais e datasheets que produzi para uma descrição mais detalhada Arise Manuais.zip


O Hardware

Os produtos de hardware da Arise permitiam que o usuário pudesse controlar a iluminação, ar condicionado e demais aparelhos da casa através de um tablet, ou programações (via PC) que deixavam a casa mais inteligente(reagindo aos ocupantes conforme regras preestabelecidas).Esse hardware não é mais comercializado, assim resolvi diponibilizá-lo no seguinte link:Arise_PCBs.zip


O Software

O software da Arise consistia de aplicações em Windows feitas em C# para a programação dos módulos e de aplicativos para o acesso do usuário, no Android e IOS. O APP para Android pode se baixado no seguinte link: RControl.apk

Simulador de voo

Esse projeto é composto de um software, feito em C#, que captura os dados de telemetria do Flight Simulator X para o envio para uma rede de atuadores conectados à uma porta serial. Um circuito com microcontrolador PIC e dois atuadores foram desenvolvidos para o projeto: o altímetro e o atuador mecânico de inclinação longitudinal e transversal da aeronave.O arquivos fonte do projeto do simulador de voo, seguem no link: SIMULADOR_FSX.zip







Emulador CHIP8

Chip8 foi uma arquitetura de máquina virtual utilizada em alguns computadores da década de 80. Esse projeto consiste de um emulador dessa arquitetura em um APP feito em Java, para o Android. O arquivo APK do projeto e os arquivos fonte seguem no link: chip8.zip










Controlador Digital

Demonstração do algoritmo de controle digital em uma simulação do controle de altitude de um foguete, atuando na potência de seus propulsores. Esse projeto está disponível em linguagem C# no link: ControleDigital.zip











Algoritmo Genético

Aplicação de teste do algoritmo genético associado a resolução de problemas de roteamento. Outros tipos de problemas podem ser resolvidos com as bibliotecas desenvolvidas nesse projeto, feito em C#. Disponível no link: Genetic.zip





Processador Neural

Este projeto consiste num processador de redes neurais para o uso em FPGAs Altera/Intel. O processador inclui instruções para o processamento do algoritmo forwardpropagation e backpropagation, e pode trabalhar com neurônios com funções de ativação ReLU (rectified linear unit). O processador opera com instruções de ponto fixo, para uma operação mais rápida e menor utilização de elementos lógicos. Esse projeto foi testado numa FPGA CycloneIII, que pode suportar até 15 desses processadores neurais, operando em paralelo. O projeto completo, feito em VHDL, pode ser baixado do seguinte link:Neural.zip






Processamento de imagem em C++
